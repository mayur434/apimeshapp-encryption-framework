# CLAUDE.md

## Project

Adobe API Mesh encryption framework ‚Äî wraps upstream GraphQL and REST (OpenAPI) operations behind AES-CBC encrypted envelopes.

## Build

```bash
npm run build:mesh              # Build mesh/mesh.json from template + secrets
npm run start:mesh -- --port 5001  # Run mesh locally (port 5000 reserved on macOS)
npm run validate                # Validate config + secrets
npm test                        # Run tests
```

## Key Files

- `mesh/mesh.json` ‚Äî Built artifact (generated by build-mesh.js, DO NOT edit directly)
- `mesh/sfdc-openapi.json` ‚Äî OpenAPI schema for Salesforce REST (MUST stay as `"type": "string"`)
- `mesh/prod-secrets.yaml` / `mesh/stage-secrets.yaml` ‚Äî Per-environment secrets (YAML `key: value`, git-ignored)
- `src/config/encryption-registry.js` ‚Äî Single source of truth for encrypted wrapper operations
- `src/config/type-defs.js` ‚Äî Auto-generates GraphQL schema from registry
- `src/hooks/before-all.js` ‚Äî Request validation hook (self-contained, no imports)
- `src/resolvers/encrypted-operations.js` ‚Äî Main resolver (self-contained, no imports)
- `scripts/build-mesh.js` ‚Äî Build orchestrator with `{{PLACEHOLDER}}` interpolation
- `scripts/env-loader.js` ‚Äî Shared secrets.yaml parser
- `scripts/crypto-envelope.js` ‚Äî AES-CBC encryption library (Node.js webcrypto)
- `scripts/encrypt-payload.js` ‚Äî CLI: encrypt GraphQL payloads
- `scripts/encrypt-rest-payload.js` ‚Äî CLI: encrypt REST body payloads

| `encryptedCreateCustomer` | Adobe Commerce | GraphQL |

| `encryptedCreateLead` | Salesforce | REST (`rest-json`) |

## Required Secrets

`MESH_AES_PASSPHRASE`, `COMMERCE_GRAPHQL_ENDPOINT`, `ALLOWED_COMMERCE_HOSTS`, `SF_BEARER_TOKEN`, `SFDC_ENDPOINT`, `ALLOWED_SFDC_HOSTS`

## Constraints

- Embedded files (hook + resolver) must be self-contained ‚Äî no `require`, `import`, `window`, `eval`, or Node built-ins
- Uses Web Crypto API (`crypto.subtle`) in mesh runtime, Node.js `webcrypto` in scripts
- Three inline registries must be kept in sync: encryption-registry.js, before-all.js, encrypted-operations.js
- Prefer `globalThis.fetch` over `global.fetch`
- Never access `process.env` or `context.secrets` at mesh runtime ‚Äî use `{{PLACEHOLDER}}` for build-time injection
- OpenAPI schemas MUST use `"type": "string"` ‚Äî complex types break Ajv on Cloudflare Workers edge runtime
- REST sources use `mode: 'rest-json'` in registry; GraphQL sources omit `mode` (default)
- SF Bearer Token uses `{{SF_BEARER_TOKEN}}` placeholder ‚Äî NOT `{context.secrets.SF_BEARER_TOKEN}`
